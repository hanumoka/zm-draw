### UI 동작 상세

- **호버**: 검은 배경 pill 툴팁으로 도형 이름 표시
- **클릭**: 도형 도구 선택 → 캔버스에 그리기 모드
- **섹션 접기**: 우측 ^ 아이콘 클릭 → 토글
- **검색**: 모든 섹션의 도형을 이름으로 필터링
- **스크롤**: 패널 전체 세로 스크롤, 얇은 스크롤바

## 커넥터 동작 (Connections 상호작용)

> 분석 소스: Animation2.gif (186프레임, 2026-02-08)
> 분석 방법: 20프레임 추출 → 프레임별 UI 상태 분석

### 커넥터 4가지 타입

| # | 아이콘 | 타입 | 설명 | zm-draw |
|---|--------|------|------|---------|
| 1 | 곡선+꺾임 | **Elbowed (Orthogonal)** | 둥근 모서리의 직각 경로 커넥터 | ⚠️ 있지만 sharp (둥근 모서리 없음) |
| 2 | 부드러운 곡선 | **Curved** | S자 베지어 커브 커넥터 | ❌ 미구현 |
| 3 | 직선 | **Straight** | 직선 화살표 | ✅ |
| 4 | 노드 구조 | **Mindmap 템플릿** | 원클릭 마인드맵 구조 생성 | ❌ 미구현 (마인드맵은 별도 도형으로 존재) |

### 커넥터 생성 플로우

| 단계 | 동작 | zm-draw |
|------|------|---------|
| 1. 도구 선택 | Connections 섹션에서 타입 클릭 → **보라색 하이라이트** | ✅ 유사 |
| 2. 커서 변경 | 캔버스 위 **"+" 십자 커서** 표시 | ✅ crosshair |
| 3. 드래그 생성 | 클릭+드래그로 실시간 그리기, **노란색 반짝이 커서** 피드백 | ⚠️ 드래그 가능, 반짝이 없음 |
| 4. 자동 선택 | 마우스 놓으면 커넥터 **즉시 선택** → 플로팅 툴바 | ⚠️ 선택은 되지만 전용 툴바 없음 |
| 5. 연속 생성 | **도구 유지** 상태에서 반복 생성 가능 | ✅ |

### 선택된 커넥터 UI

**FigJam 동작:**
- 커넥터 라인 → **파란색 하이라이트**
- 양 끝 → **파란 원형 핸들** (드래그로 재연결 가능)
- 커넥터 위 → **전용 플로팅 툴바** 표시

**커넥터 전용 플로팅 툴바 구성:**

```
┌──────────────────────────────────────────────────┐
│ ⚪ │ 📐 │ T │ ── │ ╌╌ │ → │ ··· │
│ 스타일 │ 라우팅 │ 텍스트 │ 실선 │ 점선 │ 화살표 │ 더보기 │
└──────────────────────────────────────────────────┘
```

| 버튼 | 기능 | zm-draw |
|------|------|---------|
| 스타일 (원형) | 커넥터 색상/굵기 변경 | ❌ 전용 툴바 없음 (우측 패널에서만 가능) |
| 라우팅 | Elbowed/Curved/Straight 전환 | ❌ |
| 텍스트 (T) | 커넥터에 레이블 텍스트 추가 | ❌ |
| 라인 스타일 | 실선/점선/파선 전환 | ❌ (우측 패널에서만 가능) |
| 화살표 | 화살촉 종류 변경 (양방향, 없음 등) | ❌ |
| 더보기 (...) | 추가 옵션 | ❌ |

### 커넥터 라우팅 상세

| 라우팅 | FigJam | zm-draw |
|--------|--------|---------|
| **Elbowed** | 직각 꺾임 + **둥근 모서리** (border-radius ~8px) | 직각 꺾임만 (sharp corner) |
| **Curved** | **S자 베지어 곡선** (부드러운 커브) | ❌ 미구현 |
| **Straight** | 직선 | ✅ 동일 |

### 4번째 타입: Mindmap 원클릭 삽입

- Connections 4번째 아이콘 클릭 → **마인드맵 구조 자동 생성**
- 중앙 노드: "Any question or topic"
- 3개 자식 브랜치: "A concept", "An idea", "A thought"
- 각 노드 더블클릭 → **텍스트 편집 모드** (보라색 하이라이트)
- 텍스트 편집 시 **스타일 툴바** 표시: 폰트, 크기, Bold, Italic, 색상

### zm-draw 개선 포인트 (우선순위)

| 순위 | 항목 | 난이도 | 영향도 |
|------|------|--------|--------|
| 1 | **커넥터 전용 플로팅 툴바** | 중 | 높음 |
| 2 | **Elbowed 둥근 모서리** (sharp → rounded) | 낮음 | 중 |
| 3 | **Curved 베지어 라우팅** 추가 | 높음 | 높음 |
| 4 | **생성 시 노란 반짝이 커서** 피드백 | 낮음 | 낮음 |
| 5 | **Mindmap 템플릿 원클릭 삽입** (Connections 4번째) | 중 | 중 |
